# Criterios-montanha-russa
Este documento descreve os critérios de idade e tamanho para acessar uma montanha russa em nosso parque temático. Estes critérios são estabelecidos visando a segurança e o conforto dos nossos visitantes.

# Critérios de Idade
- Até 12 anos: você paga R$5,00
- Até 18 anos: Você paga R$7,00
- Maior de 18: Você paga R$12,00

# Critérios de Altura
- Acima de 120 cm você tem altura suficiente para andar na montanha russa
